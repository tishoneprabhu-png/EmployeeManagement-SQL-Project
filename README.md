🔥 Employee Management System (MySQL Project)
📌 Project Overview

This project is a relational database system built using MySQL to manage employees, departments, and attendance records. It demonstrates real-world database operations such as joins, aggregations, stored procedures, views, and triggers.

🛠️ Tech Stack
MySQL
SQL (DDL, DML, DQL)
Database Concepts (PK, FK, Constraints)
📊 Database Structure
Tables Used:
Departments → Stores department details
Employees → Stores employee information
Attendance → Tracks daily attendance
SalaryAudit → Logs salary changes
⚙️ Features
Employee and Department relationship using Foreign Key
Attendance tracking system (Present/Absent)
Salary analysis using aggregate functions
Department-wise reports using JOIN
Views for simplified data access
Stored Procedure for department-based employee filtering
Trigger to track salary changes automatically
🔍 Key Functionalities
List employees by department
Find highest and lowest salary employees
Calculate total and average salary
Attendance status filtering
Department-wise employee count and salary distribution
Salary change audit logging using triggers
▶️ How to Run
Open MySQL Workbench or any MySQL client
Run the SQL file: EmployeeManagement.sql
Execute queries step by step
Test stored procedure and trigger using sample data
